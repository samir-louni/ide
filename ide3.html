Sublim Text

Pensez différemment, tel est ce logiciel, une pépinière d'idée ! Disponible pour Mac OS X, Linux et Windows, ce logiciel est une alternative intéressante face à Notepad++. Fait assez inusité l'installateur est écrit en langage de programmation Python. Il comprend le langage de programmation que vous écrivez et il peut vous proposer des noms de fonctions ou de classes dans la mesure du possible. Il est possible aussi d'ajouter des plug-ins en Python s'intégrant très facilement. La recherche «Goto anything» permet d'avoir un aperçu live lorsqu'on tape le texte à rechercher, ainsi il fouillera automatiquement dans le texte au fur et à mesure que l'on effectue des recherches.

Les avantages

Toutes les nouveautés, rien que des nouveautés. L’éditeur propose cependant des fonctions plus avancées, dont :
Minimap : prévisualisation de tout le fichier dans une barre latérale,
Sélection et édition dans plusieurs sections de code en parallèle (multi-curseur),
Marque-page au sein même des fichiers,
Sauvegarde automatique,
Recherche et remplacement par texte simple ou expressions régulières,
Support des macros et de greffons écrits en Python,
Personnalisation des raccourcis clavier.

A savoir

Sublime Text est un éditeur de texte et de code source, disponible sur de multiples plates-formes. Il dispose d’une API utilisant Python. Sublime Text est un logiciel propriétaire, soumis à l’acceptation d’une licence, mais est entièrement gratuit. La licence (gratuite) est à durée indéterminée, avec seulement des rappels occasionnels proposant d’acheter une inutile licence commerciale (pour 20$). Sublime Text intègre la plupart des fonctionnalités de base d’un éditeur de texte, dont la coloration syntaxique personnalisable, l’auto-complétion, un système de plugins…
